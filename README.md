# yawdigitalweb
 
## Development setup

# Install Eleventy
npm install @11ty/eleventy

# Run dev environment, serve locally and watch CSS changes
npm run dev


## Command line usage
# Searches the current directory, outputs to ./_site
npx @11ty/eleventy

# Add a web server to apply changes and refresh automatically. 
npx @11ty/eleventy --serve  

# Watch tailwind CSS changes
npm run css
